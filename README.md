# QR 팝업

## 소개

본 확장은 Context Menu에 등록된 `/qr-set` 명령을 실행할 때, 해당 QR 세트를 메인 채팅창이 아닌 **독립적인 팝업 창**으로 띄워줍니다. 이를 통해 QR 세트 버튼이 채팅 화면을 밀어올리는 일 없이, 화면의 어느 위치에서든 빠르고 편리하게 사용할 수 있습니다.

본 확장의 기능 및 코드를 참고하여 게시글 또는 제작에 활용하실 경우, 다음 정보를 반드시 명시해 주십시오: 확장명(QR팝업)과 출처 링크(해당 확장팩을 공유하거나 설명한 원제작자 게시글의 링크 (GitHub 링크가 아닌, 확장을 공유한 페이지 링크를 명시해 주시면 됩니다.)

## ✨ 주요 기능 및 장점

### 1. 독립적인 QR 세트 팝업

* **자유로운 배치:** QR 세트 목록이 채팅창 내부를 가리지 않는 독립적인 팝업 창으로 표시되어, 화면의 어느 곳이든 마우스로 **드래그하여 원하는 위치에 배치**할 수 있습니다.
* **위치 및 크기 저장:** 팝업의 위치와 크기가 자동으로 저장되어, 다음에 팝업을 열 때 이전에 사용했던 상태로 복원됩니다.

<br  />

### 2. 다양한 커스터마이징 및 제어 기능

* **테마 색상 변경:** 팝업 헤더와 스크롤바에 적용되는 **테마 색상**을 설정에서 자유롭게 변경할 수 있습니다.
* **팝업 위치 초기화:** 버튼 한 번으로 팝업의 위치를 **화면 중앙**으로 즉시 초기화할 수 있습니다.
* **수동 크기 고정 모드:**
    * 이 모드를 켜면 팝업의 우측 하단 리사이징(드래그)을 통한 **크기 조절이 불가능**해지고, 설정된 **가로/세로** 값으로 크기가 고정됩니다.
    * 크기 고정 시, 입력 필드의 값이 즉시 팝업에 반영됩니다.

<br  />

### 3. 모바일 레이아웃 모드 지원

* **모바일 지원:** '모바일 레이아웃 모드'를 활성화하면 팝업이 채팅창 너비에 맞춰 **화면 중앙에 고정**됩니다.
* **유동적인 크기:** 이 모드에서는 팝업의 위치/크기가 모바일 유저들의 각자 채팅창 화면에 맞게 채워져 사용감을 개선하는 것을 목표로 합니다.

## 1.0.1 ~ 1.0.2 업데이트 사항 (현재 버전)

<br  />

* **모바일 레이아웃 모드 추가:** 모바일 환경에서 QR 팝업을 중앙에 고정하고 크기를 최적화하는 기능이 추가되었습니다.
* **수동 크기 고정 모드 추가:** 팝업의 크기 조절을 비활성화하고 설정된 크기로 고정하는 기능이 추가되었습니다. (`settings.lockSize`, `.no-resize` 클래스)
* **팝업 위치 초기화 기능 추가:** 팝업을 화면 중앙으로 이동시키는 버튼이 설정에 추가되었습니다.
